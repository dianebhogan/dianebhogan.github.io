Release 57 ( CORE )



Sprint Began: 6/26/2017     Sprint Ended: 7/17/2017


The following items have been completed in this release:


Enhance Act-On API - Adding Contact to List - ENHANCEMENT

New API Configuration parameter 'contactcategory=x'.  Required (if not present an exception will be logged and no records will be processed.  'x' can be 'all' for all Contact Categories or a specific Category can be entered and only Contacts of that Category will be processed.


Company Holiday table - Task Due Dates by Hour and Business Days  - ENHANCEMENT

A Company Holiday table has been added to our CORE solution - this holiday table is designed so Clients can maintain the Holiday schedule that matches their internal company calendar.

The Template task configuration tab has been enhanced to accept 'bh' (business hours) and 'bd' (business days) as valid offset attributes.  This calculation is based on the business Open and Business Close settings on the Miscellaneous tab in App Settings and takes into account the new CORE Holidays table, not including holidays in the calculated offset.  This offset also takes into account the Week Start Day / Week End Day stored in a hidden field which defaults to Monday and Friday respectively.  


Expose Notification and Homepage App Setting - ENHANCEMENT

A hidden app setting has been added to the Miscellaneous tab in App Settings which will enable the Notification feature in ClientSpace NEXT.  Contact your account manager if you would like to enable the Notification feature in your installation.  Your account manager will need to recycle the ClientSpace application pool after enabling this feature for it to take effect.


Dataform Activities Right Menu Security 

Resolved an issue in NEXT where Activities on an org would be hidden unless you had edit rights to the organization.  Activities are now visible from the Org (read-only) if you have View rights on the org.  These activities become editable if you are given edit rights to the org.


Secure Action Center on Dataforms from External Users - NEXT

Action Center visibility (right nav) has been removed from dataforms for all external users.  External users will still see links and attachments in the right nav menu.


Error Uploading File

Resolved an issue where a null reference exception was being logged related to file uploads and null byte size files.
