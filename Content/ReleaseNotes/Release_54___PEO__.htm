Release 54 ( PEO )

Sprint Began: 4/17/2017     Sprint Ended: 5/5/2017

The following items have been completed in this release:


Header Info. Widgets (NEXT)

Continued Widgetizing dataform headers - 

  • The Batch Info. on dataforms Pricing Batch, Pricing Batch State, and Pricing Comp Code .
  • The Client info on dataforms Client Service Case, Client Master, Client Rescue, Client Visit, Client Team, Surcharge, Workers Comp Policy, Pricing Batch, Pricing Batch State, Pricing Comp Code

Have all been converted to Widgets, standardizing the modular design and  allowing for more flexibility of future development


Header Billing Info Widget (NEXT)

The Billing Info panel used on the Pricing Comp Code dataform header has been re-engineered as a widget. This standardizes the modular design, allowing for more flexibility of future development.


Header Rate Info. Widget (NEXT)

The Header Rate Info on the Pricing Comp Code in NEXT has been converted to a widget to standardize  the modular design and allow for more flexibility of future development.


Header Policy Info. Widget (NEXT)

The Policy Info. on Workers Comp Policy dataform header has been converted to a widget to standardize  the modular design and allow for more flexibility of future development.

Header Case Info Widget (NEXT)

The Header Case Info on the  Client Service Case header in NEXT has been converted to a widget to standardize  the modular design and allow for more flexibility of future development.  The Case Info widget now also contains escalation information for the related case.


Surcharge Profit Info Widget (NEXT)

The Header Profit Info on the Surcharge form in NEXT has been converted to a widget to standardize  the modular design and allow for more flexibility of future development.

Employee voucher import update

The final process for the Employee Voucher import has been modified to update the Last Payroll Date in the Current Employment record to the PAYDATE of the employee voucher being imported


User having issue loading home page (NEXT)

Resolved an issue with Widgetized home page where an error was thrown if a user did not have access to one of the Home pages configured widgets when the page loads.  Page now loads without an error, but does not display the Widget for which the user has no access.


Error when trying to access Pricing console when in Implementation stage (NEXT)

Resolved an issue where attempting to open the Pricing Console from a workspace in "implementation" status would cause a javascript error and the pricing console would not open.


RE: How to delete follow-ups for "null" orgs (NEXT)

Fixed an issue on the Home Dashboard in NEXT where the followups pane would display "null" for an Organization name if that Org was marked as Inactive.  System now displays full org name regardless of whether it is Active or not
