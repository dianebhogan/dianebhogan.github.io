Release 37 ( NEXT )

Product Version: NEXT – Release 37 Notes - The following Tasks have been completed/fixed in this release.

Task 33905                                                                                                                                    

Deleting a comparison batch   

Resolution:                                                                                                    

A Delete link has been created on the Comparison batch.  This link, when clicked will delete the batch and associated state(s), and code(s) as well as deactivate the surcharges associated with the declined batch and disassociate the related tasks, attachments and CRM notes (if any).

Task 33787                                                                                                                                    

Task Complete header link  

Resolution:                                                                                                    

A workflow link has been created with an action of "Complete" that sets the status, completed by, competed date, and archive on the task.  The link should be configured with conditions for each status other than "3-Complete".

Task 33785                                                                                                                                    

NEXT - Broker External Landing Page   

Resolution:                                                                                                    

A Landing Page has been created for Brokers in NEXT, which acts as a homepage for Broker users, replacing the Broker Portal Workspace in NEXT.

Task 34103                                                                                                                                    

NEXT - Prospect Landing Page

Task 34147

Task 34091                                                                                                                                    

Wage Statement Function

Resolution:                                                                                                    

A custom function has been added to the Workers' Compensation Claim form in ClientSpace NEXT which uses the Jurisdiction State on the Claim to look up and run the appropriate Wage Statement merge for that state.  Where the State has not been configured with a merge document, the system displays the error:

"There is no merge configured for this Jurisdiction State."  For information on how these merges are configured contact your NetWise account manager.

Task 34248

Missing Landing Page for Approved RFPs

Resolution:                                                                                                    

Which Landing page is opened from the Workspace and Client Search pages is determined by the Contract Stage metadata associated with the Contract Status Lookup value of the Workspace.  This allows you to group together multiple statuses and align the workspace landing page with the appropriate portion of the pricing workflow.
