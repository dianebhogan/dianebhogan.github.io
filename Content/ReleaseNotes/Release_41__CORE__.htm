Release 41( CORE )

Product Version: CORE– Release 41 Notes - The following Tasks have been completed/fixed in this release.

Task 35428                                                                                                                                      

Calendar Invite (and .ics file)   

Resolution:                                                                                                    

The calendar invite file (.ics) produced by ClientSpace calendar events has been updated to contain just the Subject Field from the Calendar appointment in ClientSpace.  The body of the appointment has been updated to remove the word "Details" and show the organization and contact (if any) but AFTER whatever is in the body.  Removed the accepted, tentative, declined section.

Task 35482                                                                                                                                     

Cancel/Rescheduling Appointment Issues   

Resolution:                                                                                                    

Resolved several issues with ClientSpace calendars including:

  • When cancelling an appointment of any kind, it does not remove the appointment from your Outlook calendar even though you got a message that it was cancelled.
  • When rescheduling a single instance of a recurring event, it inserts, what appears to be a new event, in Outlook for the new date/time and leaves the original instance as is. 
  • When a single occurrence of of a recurring meeting is rescheduled and then later cancelled, the email notification contains a "not supported calendar message.ics" file which won't allow the appointment to be removed from Outlook.
  • When an entire series of recurring events is cancelled, the email notification contains a "not supported calendar message.ics" and leaves the entire series in Outlook. This happened in both Classic and Next

Task 35493 - ENHANCEMENT                                                                                                                                       

Status on Calendar Events   

Resolution:                                                                                                    

Calendar Events have been enhanced with the following:

  • The Created By User is displayed as the Meeting Organizer
  • A field has been added to Calendar Event called Status (lookup; required; Pending, Confirmed, Completed, Canceled)
  • A field has been added to the Calendar Event called Meeting Category.
  • A Business Rule has been added to the Calendar Event to be triggered when the Status is Completed.  This rule inserts a new Activity on the Org with the following values:
  • Subject = copy from event.
  • Activity Type = "APT-" + Calendar Meeting Type Decode value
  • Contact = Copy from Event Contact field.
  • Body = copy from event.
  • Activity Date = Event End Date.
  • Activity Result = "Appointment Completed" (new lookup value CRM Activity Result group)
  • Calendar Event = link to Calendar Event creating the Note

Task 35571                                                                                                                                      

HTML in email to case   

Resolution:                                                                                                    

Resolved an issue where under certain circumstances the email integration would retain embedded html encoding when creating a client service case.  If there is embedded html, such as table information generated from a website embedded in the email this is now appropriately removed and the text contents of the table remains.

Task 35632 - ENHANCEMENT                                                                                                                                       

Simplify turning an Izenda report or dashboard into a module in ClientSpace   

Resolution:                                                                                                    

Updated the Modules system in ClientSpace to replace backslashes in the module url with the appropriate urlencoded value.

Task 35760 - ENHANCEMENT                                                                                                                                      

Enhance the Email Integration Architecture to Process Multiple Email Addresses   

Resolution:                                                                                                    

The Email Integration Architecture has been enhanced to process multiple email addresses.  Each separate mailbox would require associated API Configuration, Email Template and Email Integration Alias.  For information on configuring Email Integration see Configuring Email Integration For Your ClientSpace Installation.

Task 35873 - ENHANCEMENT                                                                                                                                       

NEXT Link in Task Email  

Resolution:                                                                                                    

The email generated when you add or make changes to a task now has a hyperlink on the Task number which will open the record in NEXT when clicked.
