Task 36231                                                                                                                                 

Delete Underscore Rule

Updated system to allow NetWise to delete CORE underscore rules resolving a field delete issue where a rule was configured on the field and the system would not allow the developer to delete the associated rule.

Email Template triggering, Phase II   

Resolution:                                                                                                    

Email Templates have been enhanced significantly.  Some of the changes include:

  • Email Recipients - Additional functionality to allow easier recipient selection (in addition to recipients available via a stored procedure)
  • Email From Address - Additional functionality to allow easier 'From' email address selection (in addition to the 'From' email address available via a stored procedure)
  • Field Replacement available for triggering dataform fields as well as via a "merge" stored procedure.
  • Each template will have replace fields from the Dataform.  If a Merge Procedure is specified, the data from that procedure will be ADDED to the dataform fields as replaceable fields
  • Clone Template -The ability to create a new template from an existing template
  • "Publish" and "Unpublish" templates - allows administrators to remove email templates from circulation while they are being edited.

For complete documentation on configuring Triggered email templates read Configuring Email Template Triggering in ClientSpace.

Task 36280                                                                                                                                 

EmailIntegration API configuration record - Port default of 993 overwritten by zero on apply   

Resolution:                                                                                                    

Modified the API Configuration form save/apply when the API Type=Email Integration: 

  • When adding or editing, if a Host was not specified, a confirmation message will appear on save/apply 
  • When adding, the Port will default to 993 and Use Secure Connection will default to "True" 
  • If the Port is cleared, it will be saved as Port 0 (nulls are not allowed)

Also corrected a bug with 'Use Secure Connection', was always setting it to true regardless of the checkbox value

Task 36318                                                                                                                                 

Email Integration Invalid Character error   

Resolution:                                                                                                    

Resolved an issue with email integration that would allow non-printable hexidecimal characters to be inserted into the generated dataform which would then fail when opening the dataform in ClientSpace.  All of these non-printable characters are now stripped from the data prior to creating the dataform

Task 36324

Resolution:                                                                                                    

File types that are not valid in ClientSpace (see this article) were getting caught by the email integration functionality and throwing exceptions.  Attachments would be stripped off.  The system now handles this better by Zipping the file and attaching it with the original file extension in the name.

Security Settings   

Resolution:                                                                                                    

We have enhanced App settings for Administrators by exposing the Application security settings,  The settings may be found under Admin Settings | App Settings | Security and include :  

  • PasswordLength - minimum length of a user password   
  • PasswordComplexity - minimum complexity of a user password (up to 4 levels)   
  • PasswordResetDays - amount of time in days from the last password reset before a user is forced to change password (unless 'Password Never Expires' is checked)   
  • LoginAttempts - number of failed login attempts before a user account is locked   
  • LockoutDuration - amount of time in minutes before a locked user account automatically unlocks   
  • SessionExpirationMinutes - number of minutes of activity allowed before a session is automatically expired. 

 For more information on application security and  passwords read Application-level security for user access.

Task 36344                                                                                                                                 

Email integration, empty body causes exception; do not process attachments starting with 'image'   

Resolution:                                                                                                    

Resolved an issue with Email Integration where the system would produce an error when it attempted to read an email with an empty "Body" field.  Added functionality to remove embedded attachments that begin with the word "image" commonly found on social media icons in email signatures.

Task 36403  - ENHANCEMENT                                                                                                                                    

Email Attachment Upload File Size   

Resolution:                                                                                                    

Changes to the Outlook Add-In allow for larger file sizes when attaching to ClientSpace.  File size limitation has been increased from 5 mb to 10 mb.  Files are attached to CRM Notes on the dataform in question. 

Attempting to attach a file larger than 10 mb will produce the error:  "The attachment size exceeds the upload limit of 10mb, please login to ... and manually attach the file to the Organization  there." 

The Email will still be posted to ClientSpace, but attachments larger than 10 mb will be stripped.
