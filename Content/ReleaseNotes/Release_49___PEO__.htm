Release 49 ( PEO )

Product Version: PEO – Release 49 Notes - The following Tasks have been completed/fixed in this release.


Task 37497(NEXT)                                                                                                                               Click Here for the Video

Apply HideOnResize property to PEO Dashboards   

Resolution:                                                                                                    

Dashboard grids in PEO.NEXT have been re-factored to utilize the "HideOnResize" property, hiding some columns by default when opened on a mobile device or in a windowed browser.  Columns can be added once the dashboard is completely loaded. 


Task 37400                                                                                                                                      

Batch to Commission Details Relationship   

Resolution:                                                                                                    

A new parent form field has been added to the Commission Detail dataform called Pricing Batch. (fkPricingBatchID).

The Projected Commissions popup has been altered to respect the related Batch relationship.

The following PEO rules have been updated to set the fkPricingBatchID when creating new Commission Records.  

  • SetBrokerCommission  
  • CreateBrokerCommissions  
  • CreateCommission  
  • UpdateCommissionRecords

The Business Rule that sets Internal Commission, External Commission and the Net Profit after Commissions field on the Pricing Batch has been updated to display only for the Commission Records associated with that Batch.

The Pricing Breakdown on Pricing Console has been updated to show Internal Commission, External Commission and Net Profit after Commissions only for the Commission Records associated with that Batch.


Task 37595                                                                                                                                      

API Mappings for Admin Billed on a Voucher   

Resolution:                                                                                                    

The HAPAPI import has been modified to sum the AdminBilled amounts on the check - it appears this is being stored as an array with multiple values, so this enhancement sums all of the Admin billed amounts to create total AdminBilled for the check. 


Task 37598                                                                                                                                      

PEO OSHA 300A report   

Resolution:                                                                                                    

The Osha 300A report has been updated to include Overtime Hours in the total hours calculation. 


Task 37620                                                                                                                                      

Divide by Zero Error   

Resolution:                                                                                                    

Resolved an issue where the business rule HE_ValidateSubmit_PerHeadMinAdmin would fail when pricing was submitted without any employees as the code was dividing by the Total EE Count, causing a "Divide by Zero" error.  Code now checks for the presence of an employee count >1 before applying the hard error validation.
