Release 39 ( NEXT )

Product Version: NEXT– Release 39 Notes - The following Tasks have been completed/fixed in this release.

Task 34619                                                                                                                                      

Add Average Wages to WSL   

Resolution:                                                                                                    

The Client Workspace Landing Page (WSL) has been enhanced to add Average Wages to the Payroll Info panel. This is the  Annualized Gross Payroll divided by Payrolled EE's and has been placed under Annualized and before Contract.  The panel has also been made expandable to reveal the Delta between expected and actual Gross payroll. 

Task 34675                                                                                                                                      

Claim Adjuster Add New   

Resolution:                                                                                                    

The "Add New" option to the Claim Adjuster field on the Comp Claim dataform - this allows you to add an organization contact directly  to the associated Carriers Organization records from the Claim form.

Task 34612                                                                                                                                     

Declined Landing Page   

Resolution:                                                                                                    

Updated the NEXT system so that Workspaces with a Contract Stage metadata set to "Declined" ( Lookup | Metadata | Contract Stage )  will use the Prospect Landing Page in NEXT - previously these types of workspaces would default to the Blank Landing page.

Task 34802                                                                                                                                      

Pricing Console Error   

Resolution:                                                                                                    

Defensive code has been added to the pricing console SaveQuickCreateStateCode() method to catch error which might occur if the comp code id passed in cannot be located in the Comp Code Master.

Task 34243                                                                                                                                     

Configurable dataform links not refreshing the page upon return   

Resolution:                                                                                                    

Fixed an issue in NEXT where a dataform would not refresh on return to the page after changes were saved on a linked dataform.  Dataform now refreshes after returning to the main form from a linked dataform.

Task 34691                                                                                                                                     

Work Item Labels Display Inconsisistently   

Resolution:                                                                                                    

Added functionality where if a default attribute is not specified on a dataform, the Left Nav item for that form would display the text "Not Configured". When this scenario now occurs the system will display the dataform name. For example: "Collateral" or "Service Team".

Task 34661                                                                                                                                      

Link datatype validation   

Resolution:                                                                                                    

Fixed a bug in Link validation that was causing URL links to produce an error when used in Internet Explorer.

Task 34692                                                                                                                                      

Date field validation   

Resolution:                                                                                                    

System has been updated so that if you enter a date into a date field with an invalid year it will not allow you to save the value - tabbing from the field clears the date.

Task 34696                                                                                                                                     

Contact Defaults   

Resolution:                                                                                                    

Updated the NEXT "Add Contact" functionality on the Organization so that it matches the functionality of Classic. 

Task 34867                                                                                                                                      

Activity time in NEXT    

Resolution:                                                                                                    

The Date Created and Date Updated fields are now being hidden on Add - the field is empty until the initial save, so it added no value to the page by displaying the fields.

Task 34720                                                                                                                                    

Activity Follow ups   

Resolution:                                                                                                    

Resolved an issue where the Assigned to and Contact were not getting defaulted in NEXT on "Save and Followup".  The system now defaults these properly.

Task 34750

Workspace Landing Page Merges   

Resolution:                                                                                                    

Fixed a bug where the Project ID was not being passed in the merge filters when running a merge from the workspace landing page.

Task 34708       Enhancement

Wildcard search on workspace name   

Resolution:                                                                                                    

Workspace Name has been added to the % search on the Workspace Dashboard. Also added workspace name, organization and DBA into the More dropdown of additional filter parameters

Task 34753                                                                                                                                     

Default Attribute on Parent Form Dropdown   

Resolution:                                                                                                    

Resolved an issue where a parent form field would not display the correct default attribute field information if the default attribute had been changed on the linked dataform.

Task 34685                                                                                                                               

Dupe OOI records   

Resolution:                                                                                                    

Fixed a bug where if more than one Org Other Info record exists for a given Organization, the system would throw the following error:

Error Text:  Subquery returned more than 1 value. This is not permitted when the subquery follows =, !=, <, <= , >, >= or when the subquery is used as an expression.

Task 34871                                                                                                                                   

Date field defaults to 1900 when entering a 2 digit year in NEXT   

Resolution:                                                                                                    

Fixed a bug where in certain browsers, entering a 2 digit year value into a date field such as 1/2/14 would cause the date to default to 1900 - 1/2/1900 instead of 1/2/2014.

Task 34850                                                                                                                                      

Contact Detail Security   

Resolution:                                                                                                    

Resolved an issue on the contact record in NEXT where the Contact Detail page wasn't respecting the same field level security entity names as in classic, for example "CRM_fkContactUserID"

Task 34851                                                                                                                                     

Forgot Password Link/Page   

Resolution:                                                                                                    

Forgot Password/Username link is now available in NEXT. This link has retained the same functionality as in Classic and is located above the Username and Password fields on the login page.

Task 34869                                                                                                                                     

Multiform Search Bug   

Resolution:                                                                                                    

Resolved an issue where the Quick Search - Equals search was inappropriately comparing number values instead of converting those values to string for comparison.  This would cause the system to throw an Invalid Conversion error.

Task 34878                                                                                                                                      

Module Add from User Profile throws "String or Binary Data would be Truncated" error when adding module   

Resolution:                                                                                                    

Resolved an issue where adding a module to the user profile with a long module name would cause the system to throw the error  "String or Binary Data would be Truncated" and not allow the module to be added. 

Task 34891                                                                                                                                      

Duplicate Orgs Created   

Resolution:                                                                                                    

Resolved an issue where Duplicate Orgs were being created when, during save if an error occurred, the duplicate check screen would reappear - clicking Ok would add additional Org records to the save pipeline

Task 34965                                                                                                                                      

'One to One' Flag on ParentForm link not being respected by Dataform Links in NEXT 

Resolution:                                                                                                    

Resolved an issue where the 'One to One' Flag on Parent Form link was not being respected by Dataform Links in NEXT.  These links would open to a multiform list instead of directly to the appropriate form.  This issue has been resolved, so that when the link is clicked you are redirected to the appropriate form, or in the case of an 'Add', into a new, blank form.
 

Task 34990                                                                                                                                     

Can't upload files attached to activity   

Resolution:                                                                                                    

Resolved an issue where the system would throw an error when an empty row GUID was being passed to the system while adding an attachment to an unsaved activity on the Org or Contact record.  The system now pre-generates the Activity RowGuid, which is used on save, allowing the system to save the attachment properly

Task 33528                                                                                                                                     

Field Selection issue in Internet Explorer   

Resolution:             

 Fixed an intermittent bug in IE where the system would allow you to select a field (the field would be highlighted in Blue), but then would not allow the field to accept focus, so you could not type into it. 

Task 35025                                                                                                                                     

Field Display Issue   

Resolution:             

Resolved an issue where Field display configured for a Multiple Core lookup type field was not functioning.
