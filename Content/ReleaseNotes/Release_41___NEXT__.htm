Task 34857                                                                                                                                     

When a file is too large, an error appears when trying to attach   

Resolution:                                                                                                    

Resolved an issue where if a user attempted to upload a file larger than the max file size for ClientSpace(defalt 20 mb) the system would throw a generic error message.  The system now checks the "Maximum File Upload Size" application setting (not exposed in the UI) and produce an appropriate error.  If for example Maximum File Upload Size is set to 15 mb, the system would present an error that that the file exceeds the maximum file size (15 mb) Please check with their administrator.  If this application setting is empty, system will default to a Max File Size of 20mb.

Task 35351 - ENHANCEMENT                                                                                                                                         

Client Scorecard   

Resolution:                                                                                                    

Resolved an issue with server side merging which was causing the worksheet cells to not maintain their named ranges during a merge.

About the Client Scorecard merge:

The Client Scorecard will enable PEOs to quickly measure the true performance of each client and their impact on the bottom line.  Detailed transactions, including payroll system data and several key ClientSpace service modules, are exported to an Excel workbook.  An activity summary that presents gross margin, cost of service, net margin, and performance against contract expectation are compared all in one concise spreadsheet.  Detailed information that feeds the cover sheet is available on other tabs of the Excel workbook and includes:

  • Contract Info (Pricing data)
  • Payroll (Payroll Statistics)
  • Client Service (cases and associated time records)
  • WorkComp (Claims, financials and associated time records)
  • Unemployment (Claims, financials and associated time records)

The Client Scorecard gives the PEO the analytics needed to communicate value and make decisions regarding ongoing pricing and servicing each account.  All at the click of a button.

Task 35501  - ENHANCEMENT                                                                                                                                       

LinkTo: Error Handling   

Resolution:                                                                                                    

Business Intelligence linkto functionality has been enhanced.  A common mistake when using linkto to open dataform links is to accept the default formatting of the dataform ID, by default Izenda uses comma separated whole numbers (1,234) instead of integers (1234) - the comma would cause an error, the text of which was quite generic and not helpful. 

The system has been enhanced to strip the commas from the ID and test for valid integer, returning the message - Invalid Record ID: The Record ID InformationRequested is not a valid integer - should you pass in something other than an appropriate integer value.
