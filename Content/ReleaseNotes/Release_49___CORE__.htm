Release 49 ( CORE )

Product Version: CORE – Release 49 Notes - The following Tasks have been completed/fixed in this release.

Task 37394                                                                                                                                      

Add Dataform Subscribers as Email Recipient Source    

Resolution:                                                                                                    

Dataform Subscribers have been added as a recipient source on Email templates, allowing you to send triggered email updates to people who have subscribed

Task 37480                                                                                                                                      

Long Workspace Names missing Benefits/Pension/financial Services description   

Resolution:                                                                                                    

The application has been enhanced so that if an Organization has a long workspace name it will no longer be truncated at 50 characters when the Project Name is generated.  The new limitation of 100 characters has been applied.

Task 37491                                                                                                                                      

Can't add rule on Credit Authorization Form   

Resolution:                                                                                                    

The HE_RequireFieldOnFormOnTrigger method has been corrected so that the rule can be configured on a Single dataform. Previously the system would throw an error if rules of this type were configured with a single form as the triggering dataform.

Task 37594                                                                                                                                      

Adding a Contact in Classic   

Resolution:                                                                                                    

Fixed a bug in classic related to new API functionality that would cause the contact save to fail.


Task 37597                                                                                                                                      

Email Template DocuSign Merge dataform key parameter   

Resolution:                                                                                                    

Standardized the parameter passed to the DocuSign merge passing a key parameter of @Key instead of @GenKey, allowing the system to utilize existing merge procedures.


Task 36712                                                                                                                                      

Issue with formatting of email   

Resolution:                                                                                                    

Resolved an issue with Email templates where Emailed CRM activities would strip out line breaks when utilizing an email template.


Task 37617                                                                                                                                      

Email to Case, marking messages as 'seen'   

Resolution:                                                                                                    

Resolved an issue where the Email to case functionality would inadvertently mark all batched messages as read if one of the messages failed.


Task 37498 (NEXT)                                                                                                                            

Extranet Calendar Recurrence Bug   

Resolution:                                                                                                    

Resolved an issue in NEXT where the "EndByDate" for recurring calendar events created in Next did not appear to be respected by calendar in Classic.  Recurring calendar start and end dates now match in Next and Classic.

Task 37545 (NEXT)                                                                                                                                      

The Requested File was not found Error   

Resolution:                                                                                                    

Resolved issue where clicking Save on the file upload dialog from a dataform file upload field without an actual file produced a null reference error.  When then trying to open the file the user would get a message, "The Requested File was not found."  File upload dialog now requires a file be attached before processing the save.


Task 35503                                                                                                                                     

Standardize on First Column Link   

Resolution:                                                                                                    

The dashboard grids in NEXT have been enhanced with an "Edit" link in the leftmost column.  This link performs the same function as double clicking the row in the grid, or selecting the row and choosing "edit".  This is especially helpful on mobile devices where double clicking the row is not an option and provides a single click option to open the record.


Task 36762                                                                                                                                      

Required fields not being required on Report filter pages in NEXT   

Resolution:                                                                                                    

Fixed an issue in NEXT where Number and date range filter pages were not respecting the "Required" attribute, allowing the report/merge to run without a required filter being completed.


Task 37501                                                                                                                                      

Column Hiding   

Resolution:                                                                                                    

The method of hiding grid fields on load and re-size has been changed to eliminate a brief flash of all grid columns prior load/resize.


Task 37596 (NEXT)                                                                                                                                        

Report/Merge Filter Page - Parent Form datatype  

Resolution:                                                                                                    

Resolved an issue with Parent forms in NEXT where Report/Merge filters were not respecting the Workspace from which the report/merge was being run and instead retrieving a complete list of those parent forms in the system.
