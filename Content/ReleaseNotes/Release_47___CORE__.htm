Release 47 ( CORE )

Product Version: CORE – Release 47 Notes - The following Tasks have been completed/fixed in this release.

Task 37221 (NEXT)                                                                                                                                      

Time Tracking Record Type Filtering   

Resolution:                                                                                                    

Resolved an issue in NEXT where the record type list on time tracking records was displaying a list of all dataforms, not just dataforms with Time Tracking enabled.


Task 37169                                                                                                                                      

Boolean Default Attribute   

Resolution:                                                                                                    

Resolved a problem where the system would display issues during dataform save if the default attribute of the dataform were set to a checkbox or a fieldset on the form.  These datatypes are no longer available to select as Default attribute on dataforms.


Task 37170                                                                                                                                      

Changing Decimal Mask Truncates Data   

Resolution:                                                                                                    

Resolved an issue in the decimal datatype where the underlying mask value stored in the database would default to 4 decimal places on any change to a decimal field, even if the mask had not been changed.  System will now prompt on mask change, requiring a case sensitive response before proceeding to save the field.

Task 37192                                                                                                                                      

Exception logged when adding / editing recipients on Email Template Triggering   

Resolution:                                                                                                    

Resolved an issue where an exception was logged each time an email recipient was added/edited on an Email Template.


Task 37240                                                                                                                                      

Upgrade Izenda to v6.10.0.12   

Resolution:                                                                                                    

The Izenda Business Intelligence reporting application has been upgraded to version 6.10.0.12


Task 37245   - ENHANCEMENT                                                                                                                                    

Expose BI settings on App Settings    

Resolution:                                                                                                    

Functionality that allows you to hide filters when a report is locked has been exposed under app settings on a new Business Intelligence tab.


Task 37269                                                                                                                                     

Workspace delete   

Resolution:                                                                                                    

Workspace delete functionality has been removed from the Workspace Search dashboard.  For any issues involving workspace delete, please create an Extranet case. 


 

Task 37220   - ENHANCEMENT                                                                                                                                    

External Calendar Appointments   

Resolution:                                                                                                    

The NEXT calendar has been enhanced so that external events will only display their subject to the user whose external calendar generated the event.  Other system users will only see the event time block and a description of "External Event for Username"  where Username represents the name of the Invitee.
