Release 53 ( CORE )

Sprint Began: 4/17/2017     Sprint Ended: 5/5/2017


The following items have been completed in this release:


Remove CreateDate and Update Date from Activity UI (NEXT)

Create Date and Updated Date have been removed from display on the header of the activity detail. These fields are still available for reporting, just not displayed.

Add link to Form Properties on the Dataform Field Attributes Popup

Previously, from the Dataform Admin list, when you clicked "Edit Attributes", the dataform list was replaced with the field list, and Dataform Admin search criteria was lost. Clicking Edit Attributes now opens the field list in a popup. When the popup is closed the previous Dataform Admin search is preserved.

Document Index - Rebuild issue

Fixed an issue where using the scheduled process to rebuild the document index caused the index table to get removed and re-created. New process now simply deletes the contents of the index, not the table itself.

Custom Datatype issue in NEXT "More" search

Resolved an issue on certain dashboards where filters under "More" options in NEXT would produce an error when selected if the filter datatype was a custom datatype based on a Parent Form. Filter drop down now functions as expected.

Add Document Search Web Service (NEXT)

A web service has been added to allow the Document Search function from Classic to be created in ClientSpace NEXT.  This work is foundational - Document Search will be added to the application in a later release.

Disable Email to Current User

Modified the Notification logic no longer send a message to the Current User.  The only exception to this is for Calendar Events and Marketing Campaigns where the current user will still get notified.
