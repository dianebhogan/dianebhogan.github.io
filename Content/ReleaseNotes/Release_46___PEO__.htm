Release 46 ( PEO )

Product Version: PEO – Release 46 Notes - The following Tasks have been completed/fixed in this release.

Task 37093                                                                                                                                      

Custom Pricing Console Calculations   

Resolution:                                                                                                    

Resolved an issue created by changes to some of the Pricing Console calculations in previous Sprint task # 36725.

Task 37137                                                                                                                                      

Filter Broker Field on Add Mode in Pricing Module   

Resolution:                                                                                                    

Resolved an issue in NEXT where when adding a workspace through the pricing dashboard, the Primary Broker field would present an unfiltered list of Organizations, instead of just Broker orgs.

Task 37157                                                                                                                                      

Please create a Server Data Proc to add Client Status from Client Master to Client Service Cases   

Resolution:                                                                                                    

ClientStatus has been added to the PEO Client Service Case server data proc and is filled from the status field on the Client Master page.  This field is available as a template task trigger via Server Data Proc1.

Task 37138                                                                                                                                      

Client Satisfaction Color on Landing Page Issue    (NEXT)

Resolution:                                                                                                    

Resolved an issue in NEXT where the Client Satisfaction level color was not being properly set on the Workspace Landing page causing the indicator to always be black, instead of representing the appropriate Satisfaction Level color.

Task 35887                                                                                                                                      

Can't use Pricing Console on iPad mini or iPhone    (NEXT)

Resolution:                                                                                                    

Resolved multiple issues in NEXT where menu interactions on Safari browser on an iPad mini and iPhone were not working as expected, including slow expansion/collapse of the left and right nav menus.
