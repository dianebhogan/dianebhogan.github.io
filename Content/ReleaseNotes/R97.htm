Enhancements

Enhancement groups are Change in Functionality, ClientSpace Premium (ClientSpace with additional modules), Advanced Administration, and General.

Change in Functionality

Case

Enhancement
47932

Workflow Channels: Add multiple users

In the Notifications step of Workflow Channels, you can now add multiple users from the Options dialog box when selecting Specific User. This enhancement saves time, allowing you to remain in the dialog until you have completed your user selections. See Configuring your workflow, Step 3: Notifications.

47502

Attachments: Multiple files

You can now drag multiple files to the Attachments bar or pane. This enhancement provides a more seamless user experience, eliminating the need to select and drag one file at a time. See Attachments.

38643

Email Add-in: Additional Contacts

When creating activities from emails, you can now add new contacts in Additional Contacts. New contacts appear in the Primary Contact and Additional Contacts lists. See Creating Activities from emails.

44172

Business Rules: Configure rules dynamically

Business rules are now more dynamic to include options for triggering. This enhancement enables you to trigger rules from various situations. Each trigger set has a trigger field, an operator field, and a value. See Configuring rules and business logic.

47467

Workflow Channels: Workflow item deletion with dependent condition

When attempting to delete a workflow item with a dependent condition in a different workflow, in the same tree, but not a descendant, an error message now displays to include a link to the dependent items. See Workflow Channel Action Items.

47140

DocuSign API: Prevent voided files from coming back to ClientSpace

A new API parameter, ExcludeVoidedFiles, allows you to prevent voided DocuSign documents from coming back into ClientSpace. See Configuring the API record.

47365

Email Admin: More search

From the Email Admin dashboard, in More search, you can now search for Workspace. For more information about the Email Admin dashboard, see Email Admin.

Business Intelligence Admin: Dashboard enhancements

The Business Intelligence Admin dashboard has new functionality, including enhanced search capabilities, Report Set Type column, and Design/Edit and Run Report buttons. These enhancements provide convenience directly from the dashboard. See Business Intelligence Admin.

Sales Module: Mobile - Add an Organization

Advanced Administration

Case

Enhancement
41548

Workflow History

A Workflow History link is now available on a dataform or task created from a Workflow Channel. This enhancement allows administrators to track the events. The link is only available to Global Admins and Developer users. See Action Center and Workflow History.

46379
47565

Scheduled Process: External Source Linking Cleanup

When deleting dataforms and workspaces, the scheduled process External Source Linking Cleanup now deletes any associated external source linking records to prevent orphaned entries. A warning message is displayed when deleting dataforms and workspaces with External Source Linking records. See Scheduled processes.

General Enhancements

Case

Enhancement
47489

ClientSpace Calendar

When viewing external calendar events with missing information, you now see a message indicating no location defined, no attendees defined, or no description. See Viewing the ClientSpace Calendar.
47366

Client Team: Loss Control Rep renamed to Risk Specialist

On the Client Team dataform, the Loss Control Rep field is now named Risk Specialist. This new title is more descriptive of the role.

46121

Tasks: Notifications changed to Subscribers

When adding tasks, the Notifications label is now named Subscribers. Searching for Subscribers now functions as the dataform Quick Search. More search now allows you to filter users by Department. For more information, see Adding tasks.

Technical Enhancements

Case

Enhancement
47678

User Profile: Removed option Default to Next

On the User Profile, the option Default to Next was removed. All user accounts now default to this setting, and therefore, it is no longer needed. For more information about the User Profile, see Your user profile and Adding and editing user accounts.
47362

Dataform API

When users with access to the Dataform API through tblApplicationUser, data requests return ClientSpace information. If the user does not have access, an error message is now displayed, stating the application user must be configured.

47772

PrismHR API: Ongoing imports

When the queue record fails to import into ClientSpace, we now stamp the error message onto the Queue record for the following Ongoing import types: Company, ClientTeam, Codes, and FutureChange. For the full list of logged import types, see the topic Import Queue management.

48427

Org Primary Workspace

When performing an Org save, ClientSpace sets the Primary Project ID from the list of projects that use it as fkOrganizationID if a workspace is not provided.

47844

Email Integration: Invalid characters

Any invalid special characters found in the email subject and body are now scrubbed. This process does not log an error message. For more information about the Email Integration scheduled process, see Email Integration processing.
