Release 65 ( PEO )

Sprint Began: 2/1/2018    Sprint Ended: 3/5/2018

The following items have been completed in this release:


COI Policy Drop down - ENHANCEMENT

The policy selector on the Certificate of Insurance has been updated to match the functionality of other areas in the system which currently use the "~".

  • By default, the policy selector will show only active records. Primary sort for this list will be on Policy Title with a secondary sort of Effective Date of policy.
  • The tilde ("~") may be used to include inactive policies - so if the user types “~A”, the drop down would show active & inactive policies where the policy title starts with the letter A.

The list for active records will display “Policy Title – Eff Date (Policy#)”.
The list for inactive records will display “Policy Title – Eff Date (Policy#) (inactive)”.


WC Policy Filter - ENHANCEMENT

The Workers Comp Claim dashboard has been enhanced -

  • A filter has been added for "Coverage State" allowing you to filter for policies by State
  • The "Coverage State" column has been added as a hidden column available on the list 


WC Claim Delete

The Workers' Comp Claim deletion function has been updated to include all dataforms with a direct foreign key relationship to the comp claim, including such records as Financials and Claim Deductible Bill records. As long as there are no active Claim Notes, or any attachments on the claim, it can be deleted using the Delete button on the Comp Claims dashboard.


Pricing Console Overall Pricing Matrix widget exception 

Corrected an issue in the Pricing Matrix Widget controller where the "use custom pricing and client code" properties were not being populated. This would happen only when the matrix was opened from the pricing console.


Org Primary Project not set

Fixed a bug that was causing the Org Primary Project to not be set during imports. The effect of this was that Contact matching in email integration was not working as it should, and generated cases would be attached to the default workspace for the affected Organization records.


Client Scorecard Merge Error

Resolved an issue where the PEO Client Scorecard Merge would fail due to improper decoding of the Carrier Claims Adjuster field.


Remove Add Button From Calendar Quadrant on chocolate home page   

The add button has been removed from the calendar quadrant of the Classic home page. Calendar events may still be added through activities, from the calendar dashboard in NEXT.


Employee Document Search File Limit   

Resolved an issue with the Employee Document Search dashboard where the record counts and page counts were incorrect, showing a maximum of 200 records regardless of the actual number.
