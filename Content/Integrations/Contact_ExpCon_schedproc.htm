Enabling the Export Contacts scheduled process

The scheduled process PrismHR Export Contacts is inactive by default. It sweeps through and selects Contact records with the Sync With Prism option enabled. Then it creates the Contacts in PrismHR.

To enable the scheduled process:

  1. Go to System Admin > Advanced > Scheduled Processes.
    The Scheduled Processes Admin dashboard opens.
  2. Open PrismHR Export Contacts (click Jump).
    The form opens.
  3. Determine the frequency of when to run the process. Work with your representative to help determine an appropriate frequency.

Frequency

Specify how many times to run this process. This is a numeric field.

Frequency Type

Select a type: Daily, Hourly, Minute, Monthly, Weekly, Yearly.

Next Run Date

Use the date picker to select the next run date.

Next Run Time

Specify the time of day.

Valid Days

Select a day of the week.
  1. If Is Active is disabled (cleared), select the Is Active option.
  2. Click Save.
