Enabling the Import Map Header

To enable the import of Contacts from PrismHR to ClientSpace, you must enable the Import Map Header, HRPAPI_contact*.csv.

  1. Go to System Admin > Advanced > Configure Import.
    The Import Map Header Search dashboard opens.
  2. Open HRPAPI_contact*.csv (click Jump).
    The Import Map Header Detail form opens.
  3. Select Active to activate.
  4. Click Save.
