Step 1: Initial

To configure a DocuSign email template:

Go to System Admin  > Email Templates.

The Email Templates dashboard opens.

Click Add.

The Add Email Template form opens to Step 1 Initial.

Complete the form.

Type

Select DocuSign. This action enables the API Configuration field.

Triggering Form

This is where your trigger originates. Select the form on which you added the Workflow link, such as Client Master. (The Workflow link was added in a previous step, Configuring the Workflow Link, in the Location field.)

Template Code

Type DocuSign.

Template Name

Type DocuSign.

API Configuration

Select DocuSign.

Click Apply.

Click Next.

The Step 2: Content form opens.
