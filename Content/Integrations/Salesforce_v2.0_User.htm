Configuring the Salesforce username

Configure the Salesforce user in ClientSpace.

To configure the Salesforce user:

  1. Go to System Admin > Users.
    The Users dashboard opens.
  2. Click Add.
    The User Details form opens.
  3. Complete the following fields:

Username

Type sfservice.

First

Type Salesforce.

Last

Type Service User.

Email

Enter the Admin Email address for ClientSpace (Go to System Admin > Advanced > App Settings).

User Type

Select External.

Authentication Type

Select ClientSpace.

Default Workspace

Select Admin Workspace.

Active

Select this option to enable the username.
  1. Click Save.
