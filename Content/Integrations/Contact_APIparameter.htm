Adding the Contact API parameter

To synchronize the Import process, you must add the ContactImportBehavior parameter.

To add the parameter:

  1. Go to System Admin > Advanced > API Configuration.
    The API Configuration dashboard opens.
  2. Open PrismHRAPI (click Jump).
    The PrismHRAPI API Configuration form opens.
  3. Click Add Parameter.
    A new row opens.
  4. In Parameter, type ContactImportBehavior.
  5. In Value, type one of the following:
    • Insert: Inserts Contacts into from PrismHR if they do not already exist.
    • Update: Updates matched Contacts in when the Contact in PrismHR has been updated.
    • Both: Performs both Insert and Update actions.
  6. Click Save.
