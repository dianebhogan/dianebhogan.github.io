Salesforce configuration

You must configure Salesforce before implementing this process. The Salesforce Partner WSDL is used because it is generic across Salesforce systems. The Client’s Salesforce account must support this API.

Request the following from the Client:

API Username

API Password

API Integration key Partner

WSDL Url

Sandbox API

Url Production API Url

Salesforce Web Interface login credentials (for testing and configuration)
