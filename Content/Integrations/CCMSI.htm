CCMSI

provides a connection to the CCMSI API, which allows you to enter the data into and export it to CCMSI. correlates the Workers' Comp Claim data and exports it to CCMSI, providing the convenience of maintaining the data in one place.

NOTE:  You must be a CCMSI customer to use the API.

 

To get started, you need the following information from CCMSI:

  • CCMSI Username
  • CCMSI Password
  • CCMSI secondary ID
  • URL that points to your organization's API services on CCMSI
