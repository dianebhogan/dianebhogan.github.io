Salesforce API overview

ClientSpace provides a customizable Salesforce integration through the Salesforce API, enabling you to connect standard (as well as custom) Salesforce fields to . This basically creates a synchronization that keeps your Salesforce and data on the same page automatically. Customizable triggering fields can be set to only sync up the data when it fits your business need.

This topic provides an overview of the functionality. Any Salesforce API implementation would be customized for you by PrismHR. If you have specific questions about the Salesforce API or would like to work with us to create a custom Salesforce integration, log an Extranet case.
