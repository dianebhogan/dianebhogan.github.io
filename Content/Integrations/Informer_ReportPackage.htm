Importing report packages in Informer

Informer is a web-based reporting utility that provides quick and easy access to data stored in multivalued or SQL-based databases. Packages offer a method of easily transporting data (reports, mappings, security settings) from one Informer instance to another.

Name: Name of the package to easily identify contents.

Last Modified: Last time package contents were changed.

Force Overwrite: Shows whether a user must accept changes in the package.

The download link puts the package on a drive where it can be imported from.

Alternatively, you can create a link that you can mail or use that will download the package.

After a package is exported to your hard drive, you can then import it into another instance of Informer.

To import a package:

Open the targeted Instance.

Go to Admin > Packages.

Click Import Package.


The Import Package dialog box opens.


Select either the downloaded file or enter the URL.

Click Next.

Navigate to Documents > Informer > NewPluginTestReports.

Select the appropriate data source. Do­ NOT select Create New Datasource.

Click Import Now.

To export individual reports:

To export individual reports, you can use the Package link in the Report template.

As well as import from the Reports Home page.
