Searching ClientSpace Help

Searching Help is easy! You can search by phrase or a term, and the results provide a list of topics containing the matching term or phrase. The search results show the topic title with the topics listed in ranked order. The results also include an excerpt from the paragraph text from the topic. This helps you decide if the page contains the information you are looking for. When you click a topic to open it, the matching terms are highlighted in the topic. To remove the highlighting, on the topic toolbar, click .

Help enables searching all files or narrowing your search using the search filter. By default, the filter is set to Search all files.

 

When you activate the filter, the selected filter displays in the Search box. The filter icon changes color to alert you that the filter is active.

To use Search:

  1. On the top banner of Help, locate the Search box.

Search

Enter a phrase or term.

Quotation marks

Use quotation marks when you want to find all occurrences of a phrase.

  • For example, user profile returns all occurrences of user profile, user, and profile.
  • To return all occurrences of the phrase, type "user profile" with the quotes.

Boolean operators

  • AND: Use to narrow the search and retrieve only the topics that contain all of the words it separates. Type AND between two or more terms. This is not case sensitive. Use quotation marks. Example: "employee" and "employment"
  • OR: Use to search for two or more words. This broadens the search results by retrieving topics that contain any of the words it separates. Use quotation marks and type OR between two or more terms. This is not case sensitive. Example: "employee" or "employment"

To narrow the search results, click and select an entry. See the example in this topic.

Color indicates the filter is active. To disable, click and select Search all files.

Click or press Enter to initiate the search.
  1. To search all Help files, type a phrase or term in the Search field, and press Enter. Use quotation marks for phrases.

To narrow your search:

  1. In the Search box, type a phrase or term.
