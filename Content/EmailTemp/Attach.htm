Adding attachments

Email Templates can have attachments that are included with the email when it is sent. Email Templates can have two different attachments:

Attach file using a file field on a dataform. If there is a file housed in the dataform field, then you can attach the file.

Attach a desktop file from the Attachment panel, using Upload File after the template is saved.

To add an attachment from a dataform:

While adding an Email Template, in Step 5 Attachments, click Add.

Select the Dataform from which the attachment will be copied.

Select the Field from the dataform selected previously that holds the attachment to be copied.

Click Finish to complete your email template configuration.

To add an attachment to a saved email template:

Go to System Admin > Email Templates.

The Email Templates dashboard opens showing a list of published Email Templates.

Open an Email Template.

Expand the Attachments panel (on the right).

Click Upload File.

The Upload File dialog box opens.

Click Choose File.

The Open dialog box opens.

Locate and select the file and click Open.

You are returned to the Upload File dialog, with the file name displayed in File.

Click Save.

The selected file now displays in the Attachments panel.
