Learning how: Assigning user templates and assuming user profiles

Configuring user profiles is performed through System Admin using the User admin module. By default, users are imported from the payroll system and inherit settings from the Client Service Associate user template. You may need to change the template, depending on the user's responsibilities.

To access user admin:

  1. Go to System Admin > Users.
    The Users dashboard opens.

  2. Locate the user and click (Open).
    The User Details form opens.

  3. Make the necessary changes and click Save.
