Using Font Awesome

Some topics use Font Awesome icons. Font Awesome stuff is in the master pages - not in topics. You should not have to do anything to make them work - other than use then as inline styling.

To implement Font Awesome icons:

  1. To add the script to a master page. (The script has been added to the master pages - so no need to do this again unless you create a new master page.)
    1. Open the topic in the Text Editor and paste the following after <head>. Copy the actual line from MasterPages/ReleaseNotes.flmsp

     

  2. In the topic, you then copy the icon from the Font Awesome site or from this topic - if it is there. You do this in the Text Editor because this is CSS. You may find a new one, or use an existing - depends on the need.
    1. For example, for the icon that represents Advanced Administration, copy the last <i> with the title Advanced Administration. You have to copy the code.
    2. The following example shows the change in functionality icon (fa-bell), premium icon (fa-plus-circle), and the system admin icon (fa-cog).

The following example shows you the code for using the icons. Font Awesome uses the <i> tag with a class. The class info comes from the Font Awesome site. You style it to fit the formatting - color and font-size. Please use the established variables for color - in the stylesheet.

Icon Code for Adv Administration

  1. Switch to Text Editor.
  2. Copy <i class="fas fa-cog" style="font-size:32px;color:var(--BrandPrimary2);padding:1px 6px 6px;" title="ClientSpace Advanced Administration" alt="ClientSpace Advanced Administration"></i>
  3. Open the targeted topic.
  4. Go to Text Editor and paste where want to display.
