ImportIDs

Certain Lookups and other data require translation to Prism values. This is done using ImportIDs on the following data:

  • CM.Contract Type
  • CM.Contract Status
  • HR Rep User
  • Sales Rep User
  • EE.Employment Status
  • EE.Employment Type
  • EE.Marital Status
  • EE.Gender
  • EE.Ethnic Classification
  • EE.Work State Filing Status
  • EE.Home State Filing Status
  • EEI.Pay Method
  • EEI.Pay Frequency
