Business Rules

Dataform

Business Rule

Clients (Client Master)
  • HE_ValidateExportClient validates that ClientSpace values are populated before allowing the Client to be added to the Export through the Scheduled Process queue.
    • HdrAction = ExportClient
  • ExportClientToHRP immediately issues this client to Prism. Updates the Export Status and Export Log fields with the results of the Prism action. If successful, sets the Client Number field with the Prism assigned value, and the Export Status to Complete.
    • HdrAction = ExportClient

Employees (Employee Demographics)
  • HE_ValidateExportEmployee validates all Prism required fields have been populated.
    • HdrAction = ExportEmployee
  • ExportToHRP immediately issues this Employee to Prism. Updates the Export Status and Export Log fields with the results of the Prism action. If successful, sets the Employee Identifier field with the Prism assigned value, and the Export Status field to EmployeeSetupPending.
    • HdrAction = EEExport
