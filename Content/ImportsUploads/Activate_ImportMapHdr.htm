Activating Import Map Headers

Some of the Import Map Headers are scripted as inactive. Therefore, to use them, you must activate the Import Map Header record.

To activate:

  1. Go to System Admin > Advanced > Configure Import.
    The Import Map Header Search dashboard opens.
  2. In More Search, select All and click Search.
    All records display to include Active and Inactive.
  3. Locate the record and click (Jump).
    The record opens.
  4. Select Active to activate the record.
  5. Click Save.
