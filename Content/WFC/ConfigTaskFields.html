Configuring task fields

Task fields are configured as other lookup type fields through System Admin > Lookups.

To configure task fields:

Go to System Admin > Lookups.

The Lookups dashboard opens.

Click More.

The More dialog box opens.

In Group, type task.

The list of task configurations becomes available.

Select an item and click Search.

If the Task Type or Workflow fields are hidden on the Task detail, it is because there are no values configured. After you add a Task Type to the lookup list, it automatically appears.
