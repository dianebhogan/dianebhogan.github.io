Setting clone-configurable dataforms

Before you create a Workflow Channel, you must determine if the channel will contain any dataform cloningCloning is when an object is generated by copying a template example of that object. Cloning is often used to provide consistency of data, or for workflow design.. If so, you need to determine which fields on the cloneable form will be available for configuration by the system.

To add fields for dataform cloning:

Go to System Admin  > Advanced > Dataform Admin.

The Dataform Admin dashboard opens.

Select the dataform you would like to clone and click Edit.

The Dataform Properties form opens.

In Features, select Enable WFC Cloning.

Click Save.
