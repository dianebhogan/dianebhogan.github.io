Viewing and publishing Workflow Channels

Workflow channels can be viewed and published by global administrators.

To view Workflow Channels:

Go to System Admin > Workflow Channels.

The Workflow Channels dashboard opens.

From the list you can do the following:

Action Description
Add

To add a workflow channel, click Add.
Delete

To delete the selected workflow channel, click Delete.
Details

To view details about a workflow channel, click Details.
Visualize

To view the item number and name, click Visualize.

To open an item in the workflow channel configuration, click an item.

Download as CSV file

You can download the workflow as a CSV file:

The file name is Workflow Channel <number>-<workflow channel name>.csv.
For example, Workflow Channel 36-Scheduled Test.csv.

The contents resemble the layout, where indentations represent children workflow items.

The workflow items and their children names are formatted as #<number>:<name>.

You can do the following:

Edit the data.

Report on the data with a graph or table.

To download the workflow as a CSV file:

From the list, select a channel (highlight the entry).

Click Visualize.

The Workflow Channel visualize dialog opens.

Click (Download as CSV file).

For Chrome, when the download finishes, you see it at the bottom of your browser.

For Internet Explorer, to view downloads, select Tools > View downloads.

Publish and Unpublish

When selected, the Published column indicates the workflow channel is published.

To unpublish the workflow channel, clear the Published option.

To publish an unpublished workflow channel, select the Published option.
View Published and Unpublished Workflow Channels 

By default, the Status chiclet is set to Published.

To toggle the Status:

Click the Status chiclet Published.

The More dialog box opens.

Select a Status: All, Published, Unpublished.

Click Search.

The list displays according to your selection.

Unpublished workflow channels display in italics and bold.

Published entries display as normal with the Published column selected.

You can also click the X beside Status Published.

This action displays all workflow channels: published and unpublished.

To reset the view to Published, click Reset.

To view the item number and name, click Visualize.
