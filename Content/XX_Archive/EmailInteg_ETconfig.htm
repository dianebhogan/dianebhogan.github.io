Configuring the Case Discussion templates

Email to Case Discussion relies on a series of non-triggered email templates: CSCDISC, CSCDISCCONFIRM, CSCDISCEMINT, and CSCDISCNEW. These templates are configured and ready for use. You can customize the template HTML for your business needs. This topic guides you through the customization.

This functionality requires several custom field replacement tags: {ReplyToURL}, {ReplyID}, {LASTREPLY_Comments}, and {Comments}. A full list of Field replacement tags that can be used in this process may be found in the topic Configuring client service case discussions.

In all cases:

Template Code MUST match what is specified here.

Template type must be System.

To configure the Case Discussion Templates:

  1. Go to System Admin > Email Templates.
    The Email Templates dashboard opens.
  2. Locate each template: CSCDISC, CSCDISCCONFIRM, CSCDISCEMINT, and CSCDISCNEW.
  3. Open the template and apply your changes:

From Email Override

The administrator can override the system default Admin Email address using the From Email Override field to override the system default Admin Email address. This allows each installation to tailor a more appropriate From email address. If you leave the field blank, the system default Admin Email is used.

{ReplyToURL}

This generates a clickable link in the email that opens the discussion in a web browser.

{ReplyID}

Only used in Subject line. This embeds the unique ID of the discussion.

{LASTREPLY_Comments}

This is replaced with the most recent discussion comments made by the external user in the thread.

{Comments}

The latest internal reply in the discussion thread.

CSCDISC

Client Service Case Discussion wrapper for outgoing discussions. This is sent to the external recipient.

CSCDISCCONFIRM

Case Reply Confirmation. Use when a case reply is received and successfully posted to the discussion thread as a new message.

CSCDISCEMINT

Case to Email Reply. This is the message to external users when a discussion response has been posted to the thread.

CSCDISCNEW

Client Service Case Discussion New Reply Template. When the external recipient sends a message, this is sent to the AssignedTo user.
