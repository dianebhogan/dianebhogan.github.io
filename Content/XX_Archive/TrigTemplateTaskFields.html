Triggering a Template Task for fields across a tabbed dataset

When triggering a template task for a field that is not on the first tab of a tabbed dataset, it is necessary to configure an "AND" group.  One condition has to be on the primary tab and the other can be the field you want to trigger on. The first tab is the parent form of the tabbed dataset.

To trigger a template task:

Determine which field on the tabbed dataform (the child dataform) you would like to trigger the task from.

Determine which field on the first tab (the parent dataform) that you can also trigger the task from.
If there is a required field, this would work best because you can trigger the task on that field being "not empty."
