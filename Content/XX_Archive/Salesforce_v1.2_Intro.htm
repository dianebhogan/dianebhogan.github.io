Salesforce API integration v1.2

This topic describes the integration of Salesforce into for Salesforce v1.2. The Salesforce API requires a large amount of precision configuration in both Salesforce and including Connectivity configuration and Import and Export mappings. Misconfiguring any of these settings can produce unexpected results and problematic consequences, such as duplicate records. Before you attempt to configure the Salesforce API integration, it is strongly recommended that you log an Extranet case for assistance.
