Adding contacts using the Outlook Add-in

You can add Contacts to the system through the Outlook Add-In.

To add new contacts:

Click + Add New Contact.

The Outlook dialog opens. From here you can add contact information.

Complete the fields:

First Name Contact first name (required)
Last Name Contact last name (required)
Primary Email Populated using the From email address of the selected email
Business Phone Complete as available
Organization System organization to which you would like to attach this contact
Or do it in ClientSpace This link allows you to open the application and create the contact directly in the system.

Click Create Contact.
You are rewarded with a success message.
