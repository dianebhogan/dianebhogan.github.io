Determining the Calendar Events Workspace

allows for multiple CRM related workspaces. This configuration is beneficial if a client wants to separate their sales group into multiple territories or divisions and it is necessary to keep their Organizations, Contacts, Calendar Events, and Activities separated.

The order of operations to determine which Workspace the calendar event is attached, is as follows:

checks to see if there is only one Client Workspace for this Org. If so, use it. If there are more than one, it prompts the user to select a workspace.

If there are Client Workspaces for this Org, it then checks if there is only one workspace that is marked as CRM Related (on the Workspace Settings tab). If so, use it.

If there are CRM Related workspaces, it then falls back to what's configured as the Sales Workspace App Setting.
