Ensuring Add State & Code is available from the Pricing Console

The Add State & Code button on the Pricing Console is secured for selected users. When the status of the Pricing Batch is set to Active (activated), you want to ensure that only certain users can add states and codes. For a list of security entities, see Standard security entities.

If the button for Quick Add does not appear for a user, check the following:

Is the user in a role with the Quick Edit entity and Add rights? 

Is the Pricing Batch in a status of New?

On the Client Master ensure that HRO/PEO Quote (Use Pricing) is set to Yes.

On the Client Master ensure that Build Pricing From Employee Census is set to No.
