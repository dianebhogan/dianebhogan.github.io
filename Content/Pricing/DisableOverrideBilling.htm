Disabling Override Billing Adjustment

This option is on the Pricing State and it is turned on by default. For the default WC Discount to work, you need to configure this field to be cleared.

To disable Override Billing Adjustment:

Go to System Admin  > Advanced > Dataform Admin.

The Dataform Admin dashboard opens.

Select Workers Comp Policy.

The Dataform Properties: Workers Comp Policy form opens.

In the Action Center, select Fields.

The list of fields displays.

Move Pass Through by Default out of the hidden fieldset and into the Policy Details.  

Configure Dataform Rows so it is at the end of the fieldset.

Go to Admin Workspace > Risk > Workers Comp Policy.

Select a policy from the list.

For Pass Through by Default, select Yes.

Any pricing states generated using this policy now defaults the Override Billing Adjustment option as cleared and the system then uses the Default WC Rate.
