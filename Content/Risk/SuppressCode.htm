Suppressing an OSHA Code

If you do not want an OSHA Code to show on the report, clear the Show on OSHA 300 Report option.

When you do not want an OSHA Code on the report:

  1. Navigate to the Admin Workspace.
    The admin tiles display.
  2. Click OSHA Code.
    The OSHA Code dashboard opens.
  3. Select the code in question and click (Open).
    The OSHA Code form opens.

  1. Clear the Show on OSHA 300 Report option.
  2. Click Save.
