User modules

User modules are shortcuts that you can add to the modules bar, which is located at the top of the workspace. The modules bar is configurable so you can add, remove, and reorder your preferences. When the width of the modules selected exceeds the amount of available space, any leftover modules overflow and are available from the View menu beside the module buttons.

When you select a module, such as Workspace, a search dashboard opens. For more details, see Search dashboards.
